## Summary

The 11% from 19% claim is directionally supported by Menlo Ventures for U.S. enterprise foundation-model API market share, but it is not a clean measure of all enterprise AI spend. The broader open-source narrative is also real in developer/startup/open-weight usage: Hugging Face and OpenRouter show rapid open-model growth. The reconciliation is that closed models dominate enterprise API dollars and regulated/mature procurement, while open models are growing in developer ecosystems, startups, cost-sensitive workloads, and self-hosted/specialized production.
