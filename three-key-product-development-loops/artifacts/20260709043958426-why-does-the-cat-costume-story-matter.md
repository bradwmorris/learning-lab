# Why does the cat costume story matter?

- Type: question

## Summary

The cat costume example shows the developer feedback loop. The coding agent can execute the feature, but the human supplies context about the user: the child loves cats, costumes may motivate practice, and the parent flow needs to fit family use. This is Andrew Ng's point that humans still matter because they hold context the AI does not have.

The cat costume story is the cleanest example of the developer feedback loop. The agent can build whatever the spec says, but it does not know the real user: his daughter likes cats, unlockable costumes might make typing practice emotionally sticky, and the grown-up login flow has to match how a parent would actually steer learning. That is the human context advantage. It is not just taste as decoration; it changes what the product should be.
